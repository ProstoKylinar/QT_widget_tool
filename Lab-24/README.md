## Библиотека Qt: сетевое взаимодействие

### Ответвление
Необходимо создать ответвление (fork) моего репозитория Lab-24 (название 
оставить без изменения, а видимость поставить приватной). Назничить меня в 
новом репозитории соавтором с правами администратора. Вики, Задачи а также
Проекты отключить.

В локальной копии создавать ветки (branch) по необходимости.

### Этап разработки
Разработать программу с использованием библиотеки Qt. В окне программы 
реализовать главное меню с пунктами "Файл" и "Помощь". В меню "Файл" добавить 
пункты "Запустить сервер" и "Остановить сервер", а также "Открыть изображение" 
и "Выход". 

При выборе "Открыть изображение" появляется диалоговое окно для открытия 
файла, который будет отправляться сервером в ответ на HTTP-запрос "GET /". 
При открытии изображения его содержимое отображается на 2/3 окна.

В меню "Помощь" добавить пункт "О программе" выводящий диалоговое окно с 
информацией об афторе и версии библиотеки Qt с которой была собрана и запущена 
программа.

Для реализации сетевого взаимодействия задействовать возможности классов 
QTcpSocket и QTcpServer.

Проект создавать на основе qmake.

Исходные коды программы снабдить краткими комментариями.

UML 2.0 диаграмму разработанных классов включить в состав исходных кодов 
программы в виде файла формата PDF.
